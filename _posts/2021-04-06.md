## YFM(YAML Front Matter)
- markdown 형식 포스트의 정보를 지정.
- YAML 형태로 작성.
- markdown 상단에 아래와 같이 표시.
```md
---
title:  "velog to github.io"
excerpt: "excerpt"

categories:
  - Blog building
tags:
  - Blog
last_modified_at: 2022-04-06T08:06:00-05:00
---
# 이중 괄호를 통해 YFM 정보에 접근 가능
e.g., {{ page.title }}, {{ page.last_modified_at }}
```
